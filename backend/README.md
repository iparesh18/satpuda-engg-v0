# Backend

Admissions API built with Node.js, Express, and MongoDB.

## Setup

1. Ensure MongoDB is running locally.
2. Update `.env` if needed.

## Run

```powershell
npm install
npm run dev
```

Server runs on `http://localhost:5000`.

## Endpoints

- `GET /api/health`
- `POST /api/admissions`
- `POST /api/contacts`
